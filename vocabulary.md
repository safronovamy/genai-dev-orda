# Vocabulary

**Generative AI** - it’s a model that analyzes existing content and creates new, unique content.

**Prompt** - a natural-language explanation of a command for generative AI. There is a special area of knowledge how to write good quality propmt - prompt engineering.

**Hallucinate APIs** - it’s a case when generative AI creates plausible information about APIs that don’t exist and presents this information as true.

**Copilot** - a special AI assistant embedded in an application. It has a narrow scope of application but performs its tasks well.

**Retrieval-augmented generation (RAG)** - a method when AI generates an answer using actual information from databases or other sources.

 **AI agent** - a multistep executor based on AI that can work autonomously.

**Factual distortions** - subtle hallucinations that are difficult to notice.

**Large knowledge graphs** - a big map of facts that shows relations between entities, is used in LLMs.

**Training data** - a set of text content for the training process of an LLM.

**Text-to-image generator** - an AI model that creates an image based on text information (prompt)

**LLM (Large Language Model)** - an AI model trained on enormous volumes of text data.

**AI assistant** - an AI application, helps do certain tasks, but can’t work autonomously like AI agent.
